# AI Medical Symptom Checker 🩺

An AI-powered web application that predicts diseases based on user symptoms using Machine Learning.

## Features
- Disease Prediction
- Symptom Checker
- Flask Backend
- Machine Learning Model
- Clean UI

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
